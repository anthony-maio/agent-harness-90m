# Task

Read all files in the workspace/notes/ directory and produce a weekly AI research digest. The digest should include: a one-paragraph summary of themes, a bullet list of notable items with links or attributions, and one "thing to test this week" recommendation.

# Allowed tools

- read_file
- list_files
- write_file
- summarize

# Cannot do

- web_search
- send_email
- delete_file

# Done means

- workspace/digest.md exists
- digest includes a summary, notable items, and one recommendation
- each notable item has an attribution (author, source, or file name)
